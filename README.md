# 🏎️ F1 Tyre Degradation & Strategy Analysis

## 📌 Overview

This project analyzes Formula 1 race data using FastF1 to study tyre degradation, driver performance, and race strategies.

## 🚀 Features

* Tyre degradation modeling using linear regression
* Driver performance comparison and ranking
* Strategy recommendation based on degradation trends
* Future lap time prediction
* Interactive dashboard using Streamlit

## 🛠️ Tech Stack

* Python
* FastF1
* Pandas, NumPy
* Matplotlib, Seaborn
* Streamlit

## 📊 Key Insights

* Identifies drivers with best tyre management
* Compares compound performance
* Classifies drivers as smooth or aggressive
* Predicts lap time degradation trends

## ▶️ Run Locally

```bash
pip install -r requirements.txt
python -m streamlit run app.py
```

## 📁 Project Structure

* `app.py` → Streamlit dashboard
* `analysis.py` → Data processing & logic
